# 论文分类索引

## 精排

### 模型扩展
- [[RankMixer]]：工业 ranking 模型 scaling backbone。
- [[UniMixer]]：统一 TokenMixer / attention / FM 的 scaling module。
- [[LONGER]]：工业推荐长序列建模与 serving 优化。
- [[OneTrans]]：序列建模和非序列特征交互的统一 Transformer。
- [[HyFormer]]：长序列解码与异构特征交互的交替统一架构。

### LTR 与多目标融合
- [[EMER]]：request-wise comparative learning 的多目标 ensemble ranking。
- [[Pantheon]]：复用多任务 tower 表征并搜索 Pareto 权重的融合排序。
- [[EASQ]]：问卷满意度监督与在线 DPO 对齐。
- [[IntEL]]：intent-aware 的融合排序。
- [[UnifiedRL]]：带探索约束的 RL-MTF。
- [[BatchRL-MTF]]：session 级 offline RL 多任务融合。
- [[xMTF]]：formula-free 单调融合函数学习。

### 生成式搜索
- [[OneSearch]]：电商搜索从 multi-stage cascade 到统一生成式框架。
- [[OneSearch-V2]]：在 OneSearch 上加入 latent reasoning、自蒸馏和 TPMA-GRPO。

## 重排

### 系统配置优化
- [[AgenticRecTune]]：用多 agent 和线上 A/B 闭环优化推荐系统配置。

### 可控多目标
- [[CMR]]：用 policy hypernetwork 根据偏好权重动态生成重排模型参数。

### 调研综述
- [[推荐系统重排最新进展]]：re-ranking / slate ranking / list-wise reranking 的近期进展梳理。
